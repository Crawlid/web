  1. html이란?

    2. 웹사이트에서 화면에 표시되는 정보를 약속한 것
    HyperText Markup Language의 줄임말

    • HyperText는 단순 텍스트 이상의, 링크 등의 개념이 포함 된 텍스트
    • Markup은 꺽쇠(<, >)로 이루어진 태그를 사용하는 규격

    태그들을 이용하여 텍스트 이상의 요소를 정의하는 약속된 언어

  2. 마크업

    3. 꺽쇠안에 들어간 것을 태그라 부르며 <something> 내용 </something>처럼 작성
    꺽쇠 안에 들어가는 문자는 속성의 이름이며, 태그 내부에 들어가는 값은 속성의 값

  3. 웹 브라우징

    4. 원리

    1. 사용자가 주소(URL) 입력
    2. 어떤 IP에 접속해야 하는지 알아냄 (DNS)
    3. 브라우저가 해당 서버에 입력한 주소(URL)로 요청
    4. 서버에서 브라우저의 요청을 해석하고 실행하여 결과 데이터 전송(HTML, CSS, JS,…)
    5. 브라우저가 서버에서 받은 값을 해석하여 화면으로 출력
    6. 웹 브라우징

  4. 태그

    5. 태그마다 속성을 부여할 수 있다.
    <태그 속성="값">으로 표시 가능

    body 태그

    • <br> : 줄바꿈 태그, 닫힘이 필요 없음
    • <p> : paragraph, 문단을 구성
    • <b> : bold, 볼드체, 글씨를 진하게함, strong태그를 더 권고
    • <i> : italic, 글씨 기울임
    • <h1~6> : 제목을 표현, 숫자가 작을수록 글씨커짐, 봇(컴퓨터, 검색엔진)이 문서를 파악할 때 쓰이므로 상황에 맞게 써주는게 중요
    • <a> : 하이퍼링크를 걸어주는 태그
      • <href> : 클릭시 이동할 링크
      • <target> : 링크를 여는 방법
        • _self: 현재 페이지 (기본값)
        • _blank: 새 탭
        • _parent: 부모 페이지로, iframe 등이 사용된 환경에서 쓰임(??).
        • _top: 최상위 페이지로, iframe 등이 사용된 환경에서 쓰임(??)
        • 프레임이름: 직접 프레임이름을 명시해서 사용할 수도 있음(??)
    • <table> : 표를 만드는 태그
      • <tr> : 표의 행
      • <td> : 표의 열, <tr>의 하위에 위치
      • <thead> : 표의 제목 영역을 나타냄, <table>의 하위, <tr>의 상위에 위치
      • <tbody> : 표의 내용 영역을 나타냄, <thead>와 같은 위치
      • <th> : 제목 셀을 나타낼 때 사용, <td>태그 대신 사용됨
      • <colspan> : <td>태그에서 속성으로 사용되며 열 병합
      • <rowspan> : <td>태그에서 속성으로 사용되며 행 병합
    • <img> : 이미지를 삽입하는 태그, 속성을 이용해 경로 및 크기 지정
      • src: 이미지의 경로
      • width: 이미지 가로 크기
      • height: 이미지 세로 크기
    • <li> : list의 약자로, 목록을 만드는 태그, 단독으로 쓰이지 않으며 <ol>, <ul> 태그 내부에 들어감
      • <ol> : 태그는 번호를 메겨 순서가 있는 (ordered list) 목록을 만듬
      • <ul> : 태그는 순서없이 모양으로 (unordered list) 목록을 만듬
